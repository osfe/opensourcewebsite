Compilation sous Debian Squeeze

Sous Debian, qui est ma distribution préféré, la version de kicad disponible dans les dépôts est un peu vielle. Pour remédier à ça on peut compiler soit même la dernière version, la difficulté est quasiment nulle. Je me suis inspiré de ce que j'ai trouvé ici et je vous la retranscris en français.

Il vous faut tout d'abord vous rendre sur le site de Kicad et télécharger les dernières sources. Une fois que vous avez récupéré l'archive, vous la décompressez et la ranger bien sagement dans le répertoire de votre choix, dans mon cas c'est le répertoire /opt

  1. # aptitude source update
  2. # aptitude install build-essential checkinstall
  3. # aptitude install cmake doxygen zlib1g-dev
  4. # aptitude build-dep kicad
  5. # cd /opt/kicad
  6. # mkdir build
  7. # cd build
  8. # cmake -DKICAD_STABLE_VERSION=ON ../
  9. # make
Si la compilation se passe bien et que vous n'obtenez pas de messages d'erreur, vous pouvez alors proceder à l'installation en utilisant la commande suivante :
  1. # make install
Vous voilà désormais avec un kicad on the bleeding edge.
